* 顯示目前Git版控狀態
    > git status

* 顯示Git版控歷史
    > git log

* 加入檔案到暫存區
    > git add <你的檔案路徑>

* 提交目前暫存區的內容
    > git commit -m "你的提交訊息"

* 上傳(同步)到remote
    > git push

    > git push <remote_name> <remote_branch>

* 下載(同步)目前的雲端的版本
    > git pull

    > git pull <remote_name> <remote_branch>

    > git pull --rebase
    
    > git pull --rebase <remote_name> <remote_branch>
